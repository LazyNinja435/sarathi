# Pixel first-install: model download + LiteRT (logcat capture)

**Purpose:** Evidence for TC-GH-018 / TC-GH-008 after installing `sarathi-v0.1.0.apk` from GitHub and using **Settings → Download offline model**.

**Device (reference):** Pixel 10 Pro XL, serial `57171FDCQ00AZ7`

## How to capture

After reproducing the flow on device, from a PC with `adb`:

```powershell
adb logcat -d | findstr /i "LiteRtLmGemmaChatEngine ModelDownloadManager ModelChunkDownloader GithubReleaseClient Sarathi AndroidRuntime"
```

Save stdout to this file (replace or append), or use a narrower filter:

```powershell
adb logcat -d | findstr /i "LiteRT-LM"
```

## Expected markers (LiteRT path)

- `LiteRT-LM load ok`
- `LiteRT-LM generation start`
- `LiteRT-LM generation end`
